## Styling

```css
/* Apply styles to radio group. */
media-radio-group {
}

/* Apply styles to radio. */
media-radio {
}

/* Apply styles when radio is checked. */
media-radio[aria-checked='true'] {
}

/* Apply styles to radio check part. */
media-radio [part='check'] {
}
```
